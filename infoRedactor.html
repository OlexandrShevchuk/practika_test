Інформація про сторінку редагування
- кнопка для додавання завдань які знаходяться в текстових полях в шаблон білету. За замувчуванням в білет додається два завдання рандомно вибраних із списку завдань, збільшити кількість завдань у білеті можна за допомогою поля вводу, яке знаходиться вище поля вводу завдань.
- кнопка для видалення завдань із шаблону.
- кнопка для додавання рівня складності завдань.
+ - кнопка для додавання поля вводу завдань
- - кнопка для видалення останнього поля для вводу завдань.
Кнопки з назвою "Додати" переносять інформацію з текстових полів над кнопками у шаблон.
