Інформація про сторінку редагування
- кнопка для додавання тексту(поле зліва від кнопки) у масив(збірний список всіх завдань написаних вами), також кількість записаних завдань можна побачити вище кнопки інформації.
- кнопка для видалення тексту(завдань) із масиву.
- кнопка для додавання завдань які знаходяться в массиві в шаблон білету. За замувчуванням в білет додається два завдання рандомно вибраних із списку завдань, збільшити кількість завдань у білеті можна за допомогою поля вводу, яке знаходиться вище поля вводу завдань.
- кнопка для видалення завдань із шаблону(в масиві ці завдання залишуться).
- кнопка для додавання рівня складності завдань, за замовчуванням у всіх завдань рівень складності 1(ще в розробці)
+ - кнопка для додавання поля вводу завдань
- - кнопка для видалення останнього поля для вводу завдань.
Кнопки з назвою "Додати" переносять інформацію з текстових полів над кнопками у шаблон.
- кнопка для додавання рівня складності завдань(знаходиться ще в розробці).
